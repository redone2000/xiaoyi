Privacy Policy / 隐私政策

Effective Date / 生效日期: February 2026

1. Introduction / 简介

This privacy policy explains how the TFTP Server application handles user data and system resources.
本隐私政策说明 TFTP Server 应用如何处理用户数据和系统资源。

2. Data Collection / 数据收集

The application does not collect, store, or transmit any personal information. Files transferred via the TFTP protocol remain entirely under the user's control.
本应用不会收集、存储或传输任何个人信息。通过 TFTP 协议传输的文件完全由用户自行控制。

3. File Access / 文件访问

The application may request access to user-selected folders in order to serve files via the TFTP service. This access is limited to the folders explicitly chosen by the user.
本应用可能请求访问用户选择的文件夹，以便通过 TFTP 服务提供文件。该访问仅限于用户明确选择的文件夹。

4. Network Access / 网络访问

The application uses network connections to provide TFTP services. No personal data is transmitted beyond the scope of the file transfer initiated by the user.
本应用使用网络连接提供 TFTP 服务。除用户发起的文件传输外，不会传输任何个人数据。

5. Third-Party Services / 第三方服务

The application does not integrate with or share data with any third-party services.
本应用不集成或共享数据给任何第三方服务。

6. Changes to This Policy / 政策变更

We may update this privacy policy from time to time. Updates will be published at this URL.
我们可能会不时更新本隐私政策。更新内容将发布在此网址。
