AIChatVoice(Let's chat) Technical Support

Welcome to use our AIChatVoice chat program!

If you encounter the following problems, please refer to the following solutions:

1. The chat program cannot be started or used normally.

- Ensure that your internet connection is normal and that your device meets our minimum system requirements.

- You can try to uninstall and reinstall the chat program, or try to clear the application cache.

2. Unable to find the required chat function or page.

- Make sure you have successfully launched the application and search for the required function or page on the main screen.

3. The chat program frequently encounters errors or disconnects.

- Ensure that your internet connection is stable.

- Please ensure that your chat program is updated to the latest version.

4. No available chat times.

- Check the available chat times on the settings screen, which can be obtained by clicking "Buy at the store” and following the instructions to get more chat times.

5. The chat program cannot answer my questions.

- The chat program is developed based on AI technology and cannot answer all questions. Please adjust your question according to the program's instructions.
